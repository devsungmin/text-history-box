## 기능 명세
- 단축키를 사용해서 화면에 표시된 텍스트를 복사할 수 있다.
- 복사된 텍스트는 history box에 저장된다.
- history box는 화면에 표시된다.
- history box에서 텍스트를 클릭하면 해당 텍스트를 복사할 수 있다.
- history box는 단축키를 사용해서 화면에 표시하거나 숨길 수 있다.
- history box는 최대 20개까지 저장되며, 앱이 종료되면 초기화된다.
- history box의 데이터는 숫자를 이용한 단축키를 이용해서 복사 할 수 있다.

## 구현 가이드
- swift ui를 사용해서 구현한다.
- 단축키는 사용자가 설정할 수 있도록 하며, 기본 단축어는 commnd + Shift + H, command + Shift + 2로 한다.
- history box는 화면에 표시되며, 텍스트를 클릭하면 해당 텍스트를 복사할 수 있다.
- history box는 단축키를 사용해서 화면에 표시하거나 숨길 수 있다.
- history box는 최대 20개까지 저장되며, 앱이 종료되면 초기화된다.
- history box의 데이터는 숫자를 이용한 단축키를 이용해서 복사 할 수 있다.
- 기본적으로 한국어와 영어를 인식해서 복사가 가능해야한다.
- 화면 캡처가 정상적으로 되는 경우 화면에 캡쳐 성공 메세지를 보여준다.
- history box에서 텍스트를 단축키로 복사하면 history box는 화면에서 숨겨진다.
